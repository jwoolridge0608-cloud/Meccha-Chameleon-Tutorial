# Meccha-Chameleon-Tutorial

## Project Description
 
A responsive beginner tutorial interface for Meccha Chameleon.

## Intended User or Player

New players who are learning the game for the first time.

## User / Player Goal

Should be able to understand the game's controls and mechanics before playing for the first time.

## Inspiration Interfaces

Video Game Tutorial Screens
  - One mechanic per section
  - Images with short descriptions

Game Control Guides
  - Clear Controls Stated
  - Organized information

Creative Learning Interfaces
  - Bright colors
  - Friendly cards

## Design Choices
- Layout: Single-page tutorial with vertical scrolling
- Color: Soft green, blue, orange, pink, and beige to create a playful painting based theme
- Typography: Baloo 2 for headings and Poppins for body text
- Visual hierarchy: Large title first, colorful tutorial cards second, completion section last
- Feedback states: Buttons change color and slightly enlarge when hovered over. Cards gain a larger colored shadow when hovered
- Responsive design: Navigation stacks vertically, tutorial cards become single-column, and images resize
